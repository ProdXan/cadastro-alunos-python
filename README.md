# cadastro-alunos-python
# Sistema de Cadastro de Alunos em Python

 1 Introdução

Este relatório tem como objetivo apresentar o desenvolvimento de um sistema simples de cadastro de alunos utilizando a linguagem de programação Python e um banco de dados MySQL por meio do XAMPP. O projeto foi desenvolvido como forma de praticar conceitos básicos de programação, organização de código e integração com banco de dados.

A proposta do sistema é permitir que um usuário cadastre informações básicas de alunos, como nome, data de nascimento e CPF. Essas informações são armazenadas em um banco de dados para que possam ser consultadas posteriormente.

Durante o desenvolvimento do projeto, foram utilizados recursos básicos da linguagem Python, além da criação e manipulação de um banco de dados utilizando o phpMyAdmin.


 2 Objetivo do Projeto

O principal objetivo deste projeto foi colocar em prática alguns conceitos aprendidos em aula, como:

 Utilização da linguagem Python para desenvolver um sistema simples.
 Criação de um banco de dados para armazenar informações.
 Conexão entre Python e MySQL.
 Organização de um projeto para publicação no GitHub.

Além disso, o projeto também ajudou a entender melhor como funciona o armazenamento de dados em sistemas e como um programa pode interagir com um banco de dados.
